Instructions to use the email extractor.

***IMPORTANT***
Do NOT change the name of the folder, the .bat file, and the .py file!!!
Only change/remove the output emails file (.csv)
***************

In Outlook, move the emails with the bounced email addresses to a separate (empty) folder.
(You may use the "For Email Extraction" folder under survey@inresearch.com.sg)

Do make sure it's an empty folder.

Run the email extractor application using the ".bat" file, NOT the ".py" file, and let it load. *it may take some time to load*

*OPTIONAL: Change folder to change the output file (the file containing the emails) location.*

Select the folder with the bounced emails from the drop-down menu.

Click extract.

Done! Output file is saved with today's date.
